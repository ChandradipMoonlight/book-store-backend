## API Reference

1. ### User Service API.

   - User registration
   - User verification
   - User login
   - Forgot password
   - Reset password


2. ###Book Service API.

    - Add book
    - Get all book
    - Delete book by id
    - Update book price
    - Update book quantity
   
3. ### Cart Service API.

    - Add books to cart
    - Delete from the cart
    - Get all book from the cart
    - Update books quantity in the cart
    - Get cart book by user
   
4. ### Order Service API.

    - Place order
    - Cancel order
    - Get all orders
    - Get all orders by user
